Type Description
IZincDataFile An interface describing a datafile where the values for parameters in a MiniZinc model are stored.
IZincFile An interface describing a Zinc file. A zinc file is a collection of ZincOxide.MiniZinc.Items.IZincItem instances. A ZincOxide.MiniZinc.Items.IZincFile can be written to a stream, and has nearly all properties of a ZincOxide.MiniZinc.Items.IZincItem instance.
IZincItem An interface specifying a generic item in a ZincOxide.MiniZinc.Items.IZincFile. An item is a ZincOxide.MiniZinc.IZincElement and must support the ZincOxide.Utils.IWriteable interface: the content must be serializable to a MiniZinc file.
IZincModel An interface describing a Zinc model: a Zinc file where the parameter variables are unassigned (unless relations between the variables are expressed).
IZincRelation Documentation for this section has not yet been entered.
ZincAnnotationItem A class representing an annotation item in a ZincOxide.MiniZinc.Items.IZincFile. An annotation file contains an identifier such that the annotation of a certain item is explained by this identifier.
ZincAssignItem An assignment item in a ZincOxide.MiniZinc.Items.IZincFile. An assignment is a tuple containing the identifier to which the assignment is done together with an expression describing how the value should be calculated.
ZincConstraintItem A Zinc item that is part of a ZincOxide.MiniZinc.Items.IZincFile that describes a certain constraint. The constraint is an expression, that is typed as a boolean, that must be true in every configuration of the problem.
ZincData A zinc file that stores data about a problem instance. Such file only assigns values to the par variables.
ZincFileBase A basic implementation of a ZincOxide.MiniZinc.Items.IZincFile a file contains a number of ZincOxide.MiniZinc.Items.IZincItem instances.

ZincIncludeItem A ZincOxide.MiniZinc.Items.IZincItem that is used to include packages that define functions, etc. in other ZincOxide.MiniZinc.Items.IZincFile instances.
ZincModel A class representing a ZincModel (see ZincOxide.MiniZinc.Items.IZincModel). A model does not assign values to parameter variables directly: only relations between the parameters are expressed.
ZincOutputItem An item in a ZincOxide.MiniZinc.Items.IZincFile that describes which parts of the model are interesting and thus must be printed. The format is described by an expression.
ZincPredicateItem Documentation for this section has not yet been entered.
ZincSolveItem An item in a ZincOxide.MiniZinc.Items.IZincFile that describes what is the task that should be carried out The options are finding a solution that satisfies the hard constraints, or minimize/maximize a given expression. The solve item can be annotated as well.
ZincVarDeclItem An item in a ZincOxide.MiniZinc.Items.IZincFile that describes a variable declaration.
